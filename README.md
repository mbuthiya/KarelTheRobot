# Karel the robot
A series of Java based challenges using Karel the robot

## Getting Started 
### Prerequisites
1. Get a copy of [Karel Learn Java](http://stanford.edu/~stepp/karel/karel-the-robot-learns-java.pdf) book
to learn all the Karel commands
2. Install Java JDK 8+
3. Install Eclipse IDE and stanford Eclipse plugin

You can then run the individual .java challenges
